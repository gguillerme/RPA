# RPA


Hoje dou inicio à um projeto onde vou utilizar o bot do software para emular a interação (mandar um email) na GUI,definindo um fluxo de trabalho para executar tarefas em uma sequência lógica. Vou fazer de uma forma que sua interoperabilidade funcione na Área de Trabalho, Aplicativos Web, Softwares,API's e Banco de Dados.
Automatizando uma tarefas agora nao tão importante, mas quem sabe pra frente venha a ser um projeto empresarial. Assim acredito que será, desejo combater tarefas manuais, repetitivas e demoradas.
